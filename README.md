# 👨‍⚕️ Smartbridge Internship VIT25 – Book a Doctor Project (MERN Stack)

Welcome to the official repository for our **Smartbridge Internship** project, created by our team for **VIT 2025**. This full-stack project is a **Doctor Appointment Booking System** built using the **MERN Stack** (MongoDB, Express, React, Node.js). The repository contains frontend and backend project folders along with individual assignment submissions for each team member.
